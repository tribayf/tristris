### Controls

- Move: `←`, `↑`, `→`, `↓`
- Slam: `Space`
- Hold: `Shift`
- Pause: `P`
- Mute Track: `M`
- Skip Track: `S`

&dagger; Currently works best in latest versions of Chrome, YMMV with other browsers  :monkey:
